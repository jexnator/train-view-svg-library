# [SKI+](https://opentransportdata.swiss/en/about-us/) | Train View SVG Library

A library of SVG icons and pictograms for visualizing wagon attributes of train formations in Swiss public transport.
This library is used in the [Swiss Train Formation Visualization](https://github.com/jexnator/train-formation-view) project.
